# Release Notes

## Version 3.1

### 新功能：近似音（模糊音）

新增「近似音」功能，可幫助用戶在輸入時自動搜尋發音相近的候選詞，解決因發音混淆導致找不到正確候選字的問題。

#### 使用方式

在「偏好設定」→「進階」中勾選「使用近似音」即可啟用。

#### 支援的模糊配對

**聲母模糊：**
- ㄅ ↔ ㄆ (b/p)
- ㄍ ↔ ㄎ (g/k)
- ㄐ ↔ ㄑ (j/q)
- ㄓ ↔ ㄗ (zh/z)
- ㄔ ↔ ㄘ (ch/c)
- ㄕ ↔ ㄙ (sh/s)

**韻母模糊：**
- ㄛ ↔ ㄜ (o/e)
- ㄣ ↔ ㄥ (en/eng)

**聲調模糊：**
- 一聲 ↔ 二聲 ↔ 三聲

#### 使用範例

| 輸入 | 可找到 |
|------|--------|
| 因該 (ㄧㄣㄍㄞ) | 應該 (ㄧㄥㄍㄞ) |
| 除發 (ㄔㄨˊㄈㄚ) | 出發 (ㄔㄨㄈㄚ) |
| 四十 (ㄙˋㄕˊ) | 事實 (ㄕˋㄕˊ) |
| 營想 (ㄧㄥˊㄒㄧㄤˇ) | 影響 (ㄧㄥˇㄒㄧㄤˇ) |

### 技術變更

- 在 `BopomofoSyllable` 類別新增 `fuzzyVariants()` 方法
- 在 `McBopomofoLM` 語言模型層實作模糊音查詢
- 新增偏好設定 `fuzzyPinyinEnabled`
- 更新偏好設定 UI 新增「使用近似音」選項
