標記 `[space]` 表示按空白鍵（一聲）。

## 預期行為說明

- **有聲調**：正常 commit（按聲調鍵或 space 送出）
- **無聲調 + 新聲母**：auto-commit，自動選最高分聲調變體
- **簡拼**（純聲母連打）：auto-commit 聲母，透過 ReadingTrie 匹配
- `→` 表示該音節是被下一個聲母觸發 auto-commit 的
